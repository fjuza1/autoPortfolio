About Me

QA Tester | Manual and Automated Testing Specialist.

I am a Dedicated QA Tester with expertise in both manual and automated testing, focused on delivering exceptional software quality assurance. Skilled in designing, implementing, and executing test cases across integration, system, and user acceptance testing (UAT) phases. Proficient in manual testing for web applications, thick clients, and APIs using tools such as Postman and SoapUI. Experienced in automating backend API testing and frontend testing workflows utilizing Cypress. Adept at ensuring product quality and reliability in dynamic development environments.
